# KNeighborsClassifier_seed_model
this is my machine learning  trained model.
🌾 Wheat Seed Variety Prediction — K-Nearest Neighbors (KNN)

Overview

This project uses the K-Nearest Neighbors (KNN) algorithm to classify wheat seeds into one of three varieties (Kama, Rosa, Canadian) based on their physical measurements.

🔗 Live App: 
https://kneighborsclassifierseedmodel-pg7nf49enj9ueujtspsujf.streamlit.app/

Dataset
Source: Seeds dataset (UCI Machine Learning Repository)
Approach
Features scaled using StandardScaler (important for distance-based algorithms like KNN)
Model: KNeighborsClassifier 
Evaluated using Accuracy, Confusion Matrix, and Classification Report
Tech Stack
Python, scikit-learn, pandas, Streamlit (for the web app)
